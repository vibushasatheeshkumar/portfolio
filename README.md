# Vibusha — Portfolio

Cybersecurity portfolio site built with React, Vite, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Pushes to `main` trigger `.github/workflows/deploy.yml`, which builds the
site and deploys it to GitHub Pages. Enable Pages in the repo settings
(**Settings → Pages → Source: GitHub Actions**) after the first push.
